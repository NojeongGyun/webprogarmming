Dom요약
브라우저가 서버에 HTML문서를 요청하면, 서버는 응답에 응하여 브라우저에 HTML파일을 보내줍니다. 
브라우저는 받은 HTML 코드를 한줄씩 읽으며 각 엘리먼트와 그 관계를 확정하여 트리구조로 바꾸고, 렌더링 되어 
코드 그대로 화면 출력이 일어납니다.
이때 Dom은 각 엘리먼트를 노드로 만들고, 노드 간의 관계를 만들어 구조를 확정 해주는 역할을 합니다.
